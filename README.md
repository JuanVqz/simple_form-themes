# SimpleForm::Themes
The easiest way to install themes for SimpleForm.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'simple_form-themes'
```

And then execute:
```bash
bundle
```

Or install it yourself as:
```bash
gem install simple_form-themes
```
